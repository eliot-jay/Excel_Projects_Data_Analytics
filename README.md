# My Excel Projects for Data Analytics

This repo contains the Excel files of my Excel projects displaying basic and advanced level Excel skills. These projects were done in conjuction with Luke Barousse's Excel for Data Analytics Course. 

## Salary Dashboard
This dashboard was created to work with a dataset of data related jobs posted to various platforms. This would help potential job seekers be able to look for jobs in the world of data science.    

[Checkout my project here!](1_Salary_Dashboard)



## Salary Analysis
This project was designed to help understand what skills top employers request out of data science jobs. This allows potential job seekers to be able to dive deeper into the data so they could focus their skills to either learn more skills or know if they have the adequate skills for competetive pay.

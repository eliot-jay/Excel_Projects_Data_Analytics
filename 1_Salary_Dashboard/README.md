## Introduction

This dashboard was created using job data from various sources of job postings in the data science fields. This data could be used by potential job seekers to check to make sure pay being offered is competetive for their specific job title and location.

## Dashboard Build
### Formulas and Functions
For this project, there was a multi-criteria filtering function that I built to populate a table, that was later turned into a data chart so the user could visually see the output of the results. 

### Charts
The charts used here change as the user makes different selections. This was my first project building out a dynamic chart in this way. 

### Data Validation
This is part of the project that I have used before. This restriction is really the user interface part of the project and allows the formulas and functions to run in the background, while updating the data visualization for the user. This is important because without restrictions the user could make a selection that the formula has not accounted for - thus "breaking" the tool.
